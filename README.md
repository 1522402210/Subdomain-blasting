# Subdomain-blasting
功能：
- [x] 子域名爆破   
- [x] 接口查询
- [x] 获取网页标题
- [x] web指纹识别 
- [x] 指定端口扫描
- [x] 网页截图

## 注意 ##
- - -
此工具适用于windows。如果是Linux用户请自行下载Chrome的驱动和浏览器
扔到脚本目录，驱动下载地址：https://sites.google.com/a/chromium.org/chromedriver/downloads
指定端口扫描，不要把80和443去掉。
